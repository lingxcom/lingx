## 系统简介
灵犀模驱，核心思想是模型驱动开发，有别于一般的代码生成系统。我们不生成代码、没有MVC、没有分层、没有前后端分离，非专业人员便可以做出一般信息管理系统。可以用于所有的Web应用程序，如网站管理后台，网站会员中心，CMS，CRM，OA。

## 主要功能
> * 在线开发：在线IDE，对数据模型进行研发及管理。
> * 字典管理：研发过程中对数据字典的统一管理
> * 权限管理：常规的用户、角色、功能、菜单管理
